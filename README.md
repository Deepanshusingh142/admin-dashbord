# user-admin-panel

Django User Management System built using MVT architecture with authentication, admin dashboard, CRUD operations, and responsive UI theme.

Django-based CRUD application following the MVT (Model–View–Template) architecture(python , html css & bootstrap,js)

User registration and login with secure authentication

Admin dashboard with integrated responsive UI theme

Create, view, and delete users directly from the dashboard

Dynamic user list rendered using Django templates

Clean separation of business logic, data models, and UI
